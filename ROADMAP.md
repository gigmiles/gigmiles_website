# GigMiles Project Roadmap 🚀

This document tracks the long-term vision and future feature ideas for the GigMiles application.

## Tax Center Enhancements
- [ ] **Schedule C Preparation**: Export (PDF/CSV) functionality to prepare data for year-end tax filings.
- [ ] **Tax Optimization engine**: Intelligent suggestions comparing Standard Mileage vs. Actual Expenses.
- [ ] **Smart Notifications**: Push/Email alerts for quarterly tax deadlines (e.g., June 15th).
- [ ] **Year-End Projections**: Predictive analytics for total tax liability by December 31st.
- [ ] **Multi-Vehicle Support**: Tailored tax methods for users operating multiple vehicles with different MPG profiles.

## Dashboard & Performance
- [ ] **Advanced Efficiency Views**: Deeper breakdowns of earnings per active hour vs. total shift time.
- [ ] **Goal Tracking**: Set and track weekly/monthly net profit targets.
- [ ] **Fuel Price Integration**: Automatically fetch local gas prices based on user location.

## Technical Debt & Polish
- [ ] **Refactor Inline Styles**: Move remaining dynamic UI styles in `BoltTodaySummary.tsx` to a more robust system (e.g., CSS Variables or Tailwind arbitrary values).
- [ ] **Test Coverage**: Increase unit tests for core financial logic in `utils/calculations.ts`.
